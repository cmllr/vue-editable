vue-editable

vue-editable is a component to manipulate data values in the view without the need to develop an edit dialog.

Basic example

Use v-editable='property' to enable editing. property is the property name in $.data, it can be an nested path (see Nested example)

Example with additional attributes

Use data-type='number' to set the input type. Attributes with data- prefix will be forwarded to the input, e.g.

  • min -> data-min
  • max -> data-max
  • name -> data-name

Nested example

{
    nested:{
        obj:{
            message: 'nested Information'
        }
    }
}

Use v-editable="nested.obj.message" to map the nested structure.
